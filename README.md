# flh-agent-standards

FLH agent standards and governance.

## How To Read This Repo

- `README.md`
  Start here for the repository overview.
- `AGENTS.md`
  Use this for navigation rules and reading order.
- `ARCHITECTURE.md`
  Read this when you need the partition model and source-of-truth boundaries.
- `docs/`
  Use this for stable governance text.
- `projects/`
  Use this for GitHub Project snapshots and repo inventory.

## Purpose

This repo is the canonical home for FLH agent governance:

- skill sharing and maintenance rules
- knowledge base governance rules
- knowledge card promotion and sharing rules
- manager starter kit for department adoption
- GitHub Project tracking for `publishes/<repo>` assets
- future agent operating standards
