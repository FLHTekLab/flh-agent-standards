# flh-agent-standards

FLH agent standards and governance.

## Purpose

This repo is the canonical home for FLH agent governance:

- skill sharing and maintenance rules
- knowledge base governance rules
- GitHub Project tracking for `publishes/<repo>` assets
- future agent operating standards

## Structure

- `docs/`: human-readable governance docs
- `projects/`: project snapshots and tracked repo inventory
- `README.md`: repo entrypoint
- `AGENTS.md`: repo-local operating notes

## Current project

- GitHub Project: `flh-agent-standards`
- URL: `https://github.com/orgs/FLHTekLab/projects/1`

## Source of truth

- Policy text lives in `docs/`
- Project inventory lives in `projects/`
- The `publishes/` subtree in `myBrainFood` remains the operational source for those repos
