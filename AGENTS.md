# AGENTS.md

## Repo Purpose

This repo stores FLH agent standards, governance rules, and project tracking snapshots.

## Operating Rules

- Prefer `docs/` as the source of truth for governance text.
- Use `projects/` for GitHub Project inventory and repo-to-item mapping.
- Keep the repo public-facing and concise.
- When adding a new tracked repo, update both the GitHub Project and the local snapshot.
